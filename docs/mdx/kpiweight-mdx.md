# KPIWeight (MDX)


  Returns the weight of the specified Key Performance Indicator (KPI).  
  
## Syntax  
  
```  
  
KPIWeight(KPI_Name)  
```  
  
## Arguments  
 *KPI_Name*  
 A valid string expression that specifies the name of the KPI.  
  
## Remarks  
 The value returned is the contribution of the KPI to the parent.
