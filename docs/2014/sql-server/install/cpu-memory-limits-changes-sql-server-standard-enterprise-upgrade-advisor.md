# Changes to CPU and memory limits for SQL Server Standard and Enterprise (Upgrade Advisor)
  [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] Standard and Enterprise supports a maximum of 64 gigabytes of system memory.  
  
||  
|-|  
|**[!INCLUDE[applies](../../includes/applies-md.md)]**  [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] Native mode &#124; [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] SharePoint mode.|  
  
## Component  
 [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)]  
  
### Description  
 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Standard edition supports 128 gigabytes of system memory and [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Enterprise supports a maximum of the Operating system maximum.  
  
> [!NOTE]  
>  You may need to reconfigure your current system settings to align with the new limits.
