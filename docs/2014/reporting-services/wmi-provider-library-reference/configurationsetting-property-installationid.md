# InstallationID Property (WMI MSReportServer_ConfigurationSetting)
  This property returns a unique identifier for a specific report server instance.  
  
## Syntax  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
