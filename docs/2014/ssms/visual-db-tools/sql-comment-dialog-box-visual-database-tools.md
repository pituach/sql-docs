# SQL Comment Dialog Box (Visual Database Tools)
  This dialog box provides an editable area where you can write a detailed description of the query. To access this dialog box, open the query or view in Query and View Designer, and in the **Properties** window, select **SQL Comment**, and click the ellipsis (**â€¦**) to the right of the property field.
