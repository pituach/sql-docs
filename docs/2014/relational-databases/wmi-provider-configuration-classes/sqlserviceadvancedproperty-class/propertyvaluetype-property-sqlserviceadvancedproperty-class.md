# PropertyValueType Property (SqlServiceAdvancedProperty Class)
  Gets the data type of a property.  
  
## Syntax  
  
```  
  
object  
.PropertyValueType [= value]  
```  
  
## Parts  
 *object*  
 A [SqlServiceAdvancedProperty Class](sqlserviceadvancedproperty-class.md) object that represents an advanced property.  
  
## Property Value/Return Value  
 A `uint32` value that represents the data type of the property.
