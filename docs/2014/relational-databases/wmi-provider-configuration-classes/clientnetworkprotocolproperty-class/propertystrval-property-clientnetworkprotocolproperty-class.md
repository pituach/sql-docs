# PropertyStrVal Property (ClientNetworkProtocolProperty Class)
  Gets the string value of the current property referenced by the [PropertyIdx Property (ClientNetworkProtocolProperty Class)](clientnetworkprotocolproperty-class.md) value.  
  
## Syntax  
  
```  
  
object  
.PropertyStrVal [= value]  
```  
  
## Parts  
 *object*  
 A [ClientNetworkProtocolProperty Class](clientnetworkprotocolproperty-class.md) object that represents an attribute of the network protocol used by the [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] client.  
  
## Property Value/Return Value  
 A string value that specifies the numeric value of the current property.
