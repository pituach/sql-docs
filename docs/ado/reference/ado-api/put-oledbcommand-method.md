# put_OLEDBCommand Method
This method performs no operation and always returns S_OK.  
  
## Syntax  
  
```  
  
HRESULT put_OLEDBCommand(  
      IUnknown *pOLEDBCommand  
);  
```  
  
#### Parameters  
 *pOLEDBCommand*  
 [in] Pointer to an OLE DB Command object.
